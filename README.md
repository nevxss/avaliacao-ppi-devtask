# DevTask - Sistema de Gerenciamento de Chamados de TI

Este projeto é um ecossistema Web Full Stack desenvolvido para a Avaliação Processual da disciplina de Programação para Internet. O sistema permite a abertura e listagem de chamados operacionais de TI.

## Tema Escolhido
DevTask: Sistema de gerenciamento de tarefas e chamados de suporte técnico para equipes de TI.

## Instruções de Execução

1. Abra o terminal na pasta `server/`.
2. Instale as dependências: `npm install`
3. Inicie o servidor: `npm run dev`
4. Abra o terminal na pasta `client/`.
5. Compile o TypeScript: `tsc -w` (ou `npx tsc -w`).
6. Abra o arquivo `client/index.html` no navegador.
